# API Purchase Store

This is an API to manage inventory control and many other features of Purchase Store. Can be cloned and used as reference to similar applications of any other store

## Running

```
npm install
npm start
```

## Environment

* Node.js v8.12.0
* MySQL Community Server 8.0.14
